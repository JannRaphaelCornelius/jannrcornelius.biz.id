My Experience in Web Development Class

Reflections on my learning journey

What I enjoyed most: Learning Tailwind CSS completely changed how I approach styling. The utility-first approach seemed weird at first, but now I can't imagine going back to writing custom CSS for everything. Building responsive designs became so much faster and more intuitive.
